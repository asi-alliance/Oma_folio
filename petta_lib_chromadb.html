petta_lib_chromadb

ChromaDB-backed Long-Term Memory Library for OmegaClaw AI Agents

Python MeTTa ChromaDB

Overview

Persistent vector-embedding memory for OmegaClaw-Core agents. Wraps ChromaDB with a clean API for storing, querying, and forgetting memories.

Python API

  • remember(content, embedding, time) - Store a memory
  • query(query_embedding, k) - Find k nearest memories
  • forget_id(item_id) - Delete a memory by ID

MeTTa Interface

The lib_chromadb.metta file exposes operations as MeTTa atoms for NARS and PLN integration.

Architecture

Uses chromadb.PersistentClient with local storage. Embeddings provided externally.

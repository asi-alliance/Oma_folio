# Cognition-Commitment Separation Protocol
## Problem: Treating every idea as a committed goal
## Prescription: Separate exploration from commitment

### EXPLORATION Phase Markers:
- Queries only, no deliverable path
- No named artifact in AC1
- No timebox
- Can be abandoned without strike
- Tracked as "exploring X" not "goal G####"

### COMMITMENT Phase Markers:
- Named persistent deliverable in AC1
- Falsifiable acceptance criteria
- Timebox or cycle budget
- Pre-commit gate passed (overlap < 80%)
- Tracked as goal with strikes

### Transition Rule:
- Exploration → Commitment requires: (1) gate check PASS, (2) named artifact, (3) ACs derived from artifact type
- Cannot skip exploration phase but CAN abandon without penalty
